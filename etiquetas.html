Estas son algunas etiquetas de html

La etiqueta "b" se utiliza para poner el texto en negrita

la etiqueta "u" se utiliza para subrayar texto

la etiqueta "s" se utiliza para poner una linea en medio del texto

la etiqueta "i" se utiliza para poner el texto en cursiva

la etiqueta "cite" se utiliza para citar un texto
